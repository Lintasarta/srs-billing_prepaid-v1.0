# Scope

## In-Scope

- Default service model for new SME customers.
- Customer deposit management, including initial top-up and subsequent deposits.
- Daily charging mechanism for subscribed services.
- Billing calculation methods: Pay Per Use (PPU) and Monthly.
- Generation of invoices upon successful deposit top-up.
- Service summary and daily charging visibility for customers.
- Monthly usage summary generation.
- Validation of sufficient balance for new service subscriptions.

## Out-of-Scope

- Postpaid billing functionalities (to be covered in a separate document).
- Customer support ticketing system (mentioned but not detailed as a primary function of the billing system).
- Resource quota limitations (Prepaid is defined as an "unlimited services" model).

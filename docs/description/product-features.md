# Product Features

- **Default Service Model:** All new sign-ups will be provisioned as Prepaid customers by default.
- **Deposit Management:** Customers must make an initial deposit to activate services and top-up as needed. The system will enforce minimum deposit amounts.
- **Daily Charging:** At 08:00 (GMT+7) each day, the system will automatically debit the customer's deposit balance for the cost of services consumed in the previous 24 hours.
- **Multiple Billing Types:** Supports various billing calculations including Pay Per Use (PPU) and Monthly rates.
- **Invoice Generation:** An official invoice, including applicable taxes (PPN) and e-materai for larger sums, is generated and emailed to the customer after every successful deposit.
- **Usage Transparency:** Customers can view a summary of all active services, an estimated daily charging summary to predict costs, and a history of daily debits.
- **Balance Validation:** The system will check if a customer's deposit is sufficient before allowing them to subscribe to new services.
- **Monthly Summary:** On the first day of each month, the system generates a comprehensive summary of the previous month's usage and charges.

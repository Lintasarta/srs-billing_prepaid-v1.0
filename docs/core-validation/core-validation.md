# Core Validation

This section describes the system's capability to provide on-demand list pricing for cloud resources.

- **System Functionality:** The system provides on-demand list pricing for cloud resources utilized by clusters.
- **Resource Support:** It supports multiple types of GPUs and various storage types.

## Measurement and Aggregation

### a. What is Measured:

- CPU  
- Memory  
- GPU  
- Storage Volume  
- Load Balancer  

### b. Aggregations:

- Container  
- Pod  
- Deployment  
- StatefulSet  
- Job  
- Controller Name  
- Controller Kind  
- Label  
- Annotation  
- Namespace  
- Node  
- Cluster
